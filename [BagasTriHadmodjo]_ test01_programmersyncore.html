1. Mengapa setiap bahasa pemrograman itu terdapat framework, apa kegunaan framework ?

Jawab: Kegunaan framework adalah untuk memudahkan programmer dalam proses pengerjaan / pengembangan sebuah situs website maupun aplikasi. Selain itu membuat Source Code menjadi lebih terstruktur / rapi


2.Jelaskan mengenai fungsi dan penggunaan if else, while, while do, switch case, for, foreach?

Jawab:

  • IF ELSE = untuk eksekusi sebuah kondisi jika kondisi benar dan kondisi salah.
  • WHILE = untuk membuat statement berulang selama kondisi syarat diberikan bernilai benar. Jika kondisi bernilai syarat maka statement tidak akan diulang.
  • WHILE DO = sebuah perulangan dan memeriksa kondisinya. Jika kondisi terpenuhi, maka akan melanjutkan perulangan.
  • SWITCH CASE = untuk membuat suatu menu dan sebagai instruksi pemilihan aksi yang akan di eksekusi.
  • FOR = pernyataan pengulangan yang dikhususkan untuk pengulangan yang secara tepat mengetahui berapa kali pengulangan akan terjadi.
  • FOREACH = untuk perulangan yang datanya dalam bentuk array dan memungkinkan untuk menampilkan data dari array tanpa variabel counter.

3. Apa itu constructor, method, class, object, variabel, parameter, function, procedure ?

Jawab:

  • CONSTRUCTOR = sebuah method berfungsi menginisialisasi beberapa variabel yang akan dimiliki oleh objek.
  • METHOD = sebuah fungsi.
  • CLASS = cetakan untuk membuat objek.
  • OBJECT = suatu komponen yang memiliki bentuk fisik.
  • VARIABEL = peubah atau letak penyimpanan yang berisi beberapa kuantitas / informasi / nilai
  • PARAMETER = suatu variabel yang berguna menampung value dan dikirimkan ke dalam fungsi.
  • FUNCTION = pembuatan sub-program dan bisa di eksekusi dengan cara klik button, saat dipanggil dari kode JS dan juga otomatis dipanggil sendiri.
  • PROCEDURE = sebuah program yang terpisah dalam blok sendiri yang berfungsi sebagai subprogram.


4. Perbedaan method=”Post” dan “Get” ?

Jawab: POST adalah mengirimkan data / nilai langsung ke action untuk ditampung tanpa menampilkan data pada URL, sedangkan GET menampilkan data/nilai pada URL, kemudian akan ditampung oleh action


5. Jelaskan konsep mengenai web service dan fungsi apa yang digunakan untuk memanggil web service client ?

Jawab: sebuah sistem yang berisi sekumpulan basis data (database) dan perangkat lunak (software) / bagian dari program perangkat lunak yang diakses secara remote oleh piranti dengan perantara tertentu. Bisa diakses dimana saja dan kapan saja. Fungsi yang digunakan adalah Service Request.


6. Jelaskan mengenai MVC dan HMVC !

Jawab: MVC / Model View Controller adalah fungsinya sebagai pemisah antar data, tampilan, dan pengontrol. Sedangkan HMVC / Hierarchical Model View Controller adalah perkembangan dari MVC dan programmer akan dimudahkan dalam mengatur bagian bagian data.


7. Sebutkan fungsi curl() dalam php!

8. Bagaimana menampilkan sebuah element array dalam PHP ?

Jawab: Dengan cara memanggil nama array diikuti indeks elemen yang ingin ditampilkan, di dalam tanda kurung siku ([]).


9. Perbedaan HTTP dan HTTPS ?

Jawab: HTTP merupakan protokol standar yang digunakan untuk mengirimkan permintaan dan menerima respon antara client dan server. Namun, data yang ditransfer melalui HTTP tidak dienkripsi, sehingga data dapat dengan mudah disadap oleh pihak yang tidak berwenang. Sedangkan HTTPS adalah varian dari HTTP yang menggunakan protokol SSL/TLS (Secure Socket Layer/Transport Layer Security) untuk mengenkripsi data yang ditransfer. Enkripsi ini membuat data yang ditransfer lebih aman dari serangan dan pihak yang tidak berwenang tidak dapat membaca atau memodifikasi data yang ditransfer.


10. Jelaskan Apa itu API? sebutkan contohnya?

Jawab: API / Application Programming Interface merupakan kumpulan aturan dan protokol yang memungkinkan berbagai aplikasi untuk berkomunikasi dan bertukar data satu sama lain.

    Contoh dari API :
  • API Twitter: Twitter memiliki API yang memungkinkan pengembang untuk mengintegrasikan fungsionalitas Twitter ke dalam aplikasi mereka sendiri, seperti mengirim tweet, membaca timeline, dan mencari tweet.
  • API Google Maps: Google Maps memiliki API yang memungkinkan pengembang untuk menampilkan peta, menambahkan penanda, menampilkan rute, dan banyak lagi melalui aplikasi mereka sendiri.
  • API Facebook: Facebook juga memiliki API yang memungkinkan pengembang untuk mengakses data Facebook seperti profil pengguna, foto, dan update status.
